Introduction to C Language

C is a structured programming language developed by Dennis Ritchie in 1972 at Bell Telephone Laboratories to support the UNIX operating system. It evolved from the B language and is considered a middle-level language due to its ability to combine low-level hardware access with high-level programming constructs. C is widely used for system software and application development.

Implementation of C Language

  • Used to develop Operating Systems e.g., UNIX.
  • Developing new programming platforms such as C++, C# (C++ is used in Blockchain Technology (Bitcoin, Ethereum, etc.).
  • Development of popular web browsers such as Mozilla Firefox, Thunderbird, etc.
  • Development of various GUIs (Graphical User Interface) and IDEs (Integrated Development Environments).
  • Popular Database Management System (DBMS) like MySQL was developed using C.

Features of C Language

  • Structured Language: Supports modular programming.
  • Modular Programming: Big programs can be reduced into simpler small program blocks called modules.
  • Supports graphics.
  • Huge library functions.
  • Case-sensitive language.

Data Types in C

Data Type Used For Format Specifier
int Non-decimal numbers %d
float Decimal numbers %f
char String %s or %c

Variables in C

  • Variable name should not start with a number.
  • Variable name should not have blank space.
  • Keywords cannot be used as variable names.
  • Uppercase variable names are different from lowercase variable names.

Operators in C

  • Arithmetic operators: +, -, *, /, %
  • Relational operators: >, <, >=, <=, !=, ==
  • Assignment operator: =
  • Logical operators: &&, ||, !

Header Files in C

  • #include for standard input output function e.g., printf, scanf, etc.
  • #include for mathematical function e.g., pow, sin, cos, etc.
  • #include for graphical elements e.g., circle, rectangle, line, etc.
  • #include for string handling functions e.g., strcpy(), strlen(), strupr(), etc.

Sample Structure of C Program

        
            #include<stdio.h>
            void main ()
            {
                // block of statements
                getch ();
            }

Explanation: C program always executes from top left to bottom right, so program execution starts from the header file which is followed by the main() function.
