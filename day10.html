Introduction to QBasic

QBasic is a simple programming language developed by Microsoft in the late 1980s. It is designed for beginners to learn the basics of programming.

Basic Loops

FOR...NEXT Loop:

FOR i = 1 TO 10
    PRINT i
NEXT i

WHILE...WEND Loop:

i = 1
WHILE i <= 10
    PRINT i
    i = i + 1
WEND

DO...LOOP:

i = 1
DO
    PRINT i
    i = i + 1
LOOP WHILE i <= 10

Conditional Operators

IF...THEN...ELSE:

age = 20
IF age >= 18 THEN
    PRINT "You are an adult."
ELSE
    PRINT "You are a minor."
END IF

SELECT CASE:

grade = "A"
SELECT CASE grade
    CASE "A"
        PRINT "Excellent!"
    CASE "B"
        PRINT "Good job!"
    CASE "C"
        PRINT "Well done!"
    CASE ELSE
        PRINT "Try harder!"
END SELECT

All Operators

Arithmetic Operators:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • ^ : Exponentiation
  • MOD : Modulus (remainder)

Relational Operators:

  • = : Equal to
  • <> : Not equal to
  • < : Less than
  • > : Greater than
  • <= : Less than or equal to
  • >= : Greater than or equal to

Logical Operators:

  • AND : Logical AND
  • OR : Logical OR
  • NOT : Logical NOT

String Operators:

  • + : Concatenation (joining strings)

Structures

Program Structure:

' This is a comment
PRINT "Hello, World!"

Subroutines and Functions:

SUB...END SUB:

SUB Greet
    PRINT "Hello, there!"
END SUB

CALL Greet

FUNCTION...END FUNCTION:

FUNCTION Add(a, b)
    Add = a + b
END FUNCTION

sum = Add(5, 3)
PRINT sum

Keywords

QBasic has several reserved keywords that have specific meanings:

  • PRINT, INPUT, IF, THEN, ELSE, FOR, NEXT, WHILE, WEND, DO, LOOP
  • DIM (declare variables), SUB, END SUB, FUNCTION, END FUNCTION, SELECT CASE, END SELECT, CASE, CASE ELSE
  • GOTO, GOSUB, RETURN, END

Characters & Variables

Character Data:

name$ = "John Doe"
PRINT name$

Numeric Data:

age = 30
height = 5.9

Variable Naming:

myVar = 10
myvar = 20 ' This will overwrite the value of myVar

DIM Statement:

DIM numbers(10) AS INTEGER
DIM names$(5)

File Handling

Opening a File:

OPEN "data.txt" FOR INPUT AS #1
OPEN "data.txt" FOR OUTPUT AS #2
OPEN "data.txt" FOR APPEND AS #3

Reading from a File:

OPEN "data.txt" FOR INPUT AS #1
WHILE NOT EOF(1)
    LINE INPUT #1, line$
    PRINT line$
WEND
CLOSE #1

Writing to a File:

OPEN "data.txt" FOR OUTPUT AS #1
PRINT #1, "Hello, World!"
CLOSE #1

Appending to a File:

OPEN "data.txt" FOR APPEND AS #1
PRINT #1, "Additional line of text."
CLOSE #1
