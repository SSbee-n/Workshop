Computer Hardware Workshop

A Comprehensive Introduction to Computer Hardware for Class 6-12 Students

Day 5: Network Systems Basics

Goals:

  • Understand the basics of networking.
  • Learn about different types of networks and their components.

Motives:

  • To comprehend how computers communicate with each other.

Detailed Information:

Types of Networks:
Image of Types of Networks

LAN (Local Area Network), WAN (Wide Area Network), PAN (Personal Area Network), MAN (Metropolitan Area Network). LANs are used for small geographical areas like a single building. WANs cover larger areas, such as cities or even global connections. PANs are for individual use, typically within a range of a few meters. MANs cover city-wide areas.

Network Devices:
Image of Network Devices

Routers, switches, modems, network cables. Routers direct traffic between different networks, switches connect multiple devices within a single network, modems modulate and demodulate signals for internet access, and network cables provide physical connections between devices.

Basic Networking Concepts:
Image of IP Addresses

IP addresses, MAC addresses, subnetting. IP addresses are unique identifiers for devices on a network, enabling them to communicate. MAC addresses are hardware addresses assigned to network interfaces. Subnetting divides a network into smaller, manageable sections, improving performance and security.

Telecommunication

Activity:

Network Setup: Set up a basic LAN using routers and switches, and connect multiple computers. This hands-on activity will give students practical experience in creating a network, understanding the connections, and troubleshooting basic network issues.
