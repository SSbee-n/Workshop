A Comprehensive Introduction to Computer Hardware for Class 6-12 Students

Day 6: Introduction to Internet and Web Technologies

Goals:

  • Learn how the internet works.
  • Understand basic web technologies.

Motives:

  • To understand the global network and how web content is served.

Detailed Information:

How the Internet Works:
Image of Internet Map

The internet is a vast network of interconnected computers and devices. Data is transferred in small units called data packets. These packets travel through various networks to reach their destination. Internet Service Providers (ISPs) offer access to the internet. The Domain Name System (DNS) translates human-friendly domain names into IP addresses that computers use to identify each other.

Web Technologies:
Image of HTML5 Logo

Web technologies include HTML, CSS, and JavaScript. HTML (HyperText Markup Language) is used to structure web content. CSS (Cascading Style Sheets) is used to style and layout web pages. JavaScript is a programming language that adds interactivity to websites.

Image of CSS3 Logo
Image of JavaScript Logo
Browsers and Search Engines:
Image of Browser Usage Share

Browsers are applications used to access and view websites. Popular browsers include Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge. Search engines like Google, Bing, and Yahoo help users find information on the web by indexing and ranking web pages based on relevance.

Activity:

Create a Simple Webpage: Using HTML and CSS, guide students to create a basic webpage. This hands-on activity will introduce students to the basics of web development and allow them to apply what they've learned about HTML and CSS.
