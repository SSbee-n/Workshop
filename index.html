Computer Hardware & Networking Workshop

A Comprehensive 7-10 Day Program for Students of Class 6-12

Explore the Schedule

Workshop Schedule

  • Day 1: Introduction to Computer Hardware

    Learn the basics of computer hardware components and their functions.

    •
  • Day 2: Storage Devices and Memory

    Understand different types of storage devices and the differences between RAM and permanent storage.

    •
  • Day 3: Input and Output Devices, Inside of a computer

    Identify various input and output devices and their roles in a computer system. Learn how to assemble/disassemble a computer and understand the internal layout.

  • Day 4: Contemporary Technologies

    •
  • Day 5: Network Systems Basics

    Understand the basics of networking and learn about different types of networks.

    •
  • Day 6: Introduction to Internet and Web Technologies

    Learn how the internet works and understand basic web technologies.

    •
  • Day 7: Introduction to Network Security

    Understand the basics of network security and learn about common threats and protection methods.

    •
  • Day 8: Advanced Networking Concepts

    Explore more complex networking topics and understand advanced network systems.

    •
  • Day 9: POP vs OOP

    Learn the working of a seven-segment display and how to interface it with a computer or microcontroller.

    •
  • Day 10: Review and Project Presentation

    Review all topics covered and present individual or group projects based on what was learned.
